# 🎬 Movie Recommendation System

A content-based Movie Recommendation System built using Python, NLP techniques, and Word2Vec embeddings. The system recommends movies based on semantic similarity between movie descriptions and metadata.

## 🚀 Features

- Movie data preprocessing and cleaning
- Genre extraction and feature engineering
- NLP-based text processing
- Word2Vec embedding generation
- Similarity-based recommendations
- Weighted rating integration

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- NLTK
- Gensim (Word2Vec)
- Scikit-learn
- Jupyter Notebook

## 📂 Dataset

The project uses movie metadata containing:

- Movie Titles
- Genres
- Plot Overviews
- Vote Counts
- Vote Averages

## ⚙️ Workflow

### 1. Data Preprocessing
- Handle missing values
- Clean movie metadata
- Parse genre information

### 2. NLP Processing
- Tokenization
- Text cleaning
- Overview preprocessing

### 3. Feature Engineering
- Generate Word2Vec embeddings
- Create movie feature vectors
- Combine genre and textual information

### 4. Recommendation Generation
- Compute similarity scores
- Rank similar movies
- Return top recommendations
